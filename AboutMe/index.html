Assignment 1

JQuery , Bootstrap
การออกแบบและสร้างหน้าเว็บแบบทดสอบออนไลน์โดยใช้ jQuery และ Bootstrap นั้นมีข้อดีหลายประการที่ช่วยให้กระบวนการพัฒนามีความรวดเร็วและมีประสิทธิภาพ โดย jQuery ช่วยให้การเขียนโค้ด JavaScript ง่ายขึ้นและรองรับการโต้ตอบของผู้ใช้ได้ดี Bootstrap เป็นเฟรมเวิร์กที่ช่วยให้การออกแบบหน้าเว็บตอบสนองได้ดีบนทุกอุปกรณ์ใช้งานได้ง่าย ด้วยเครื่องมือและเทมเพลตที่เตรียมไว้ให้ ทั้งสองเครื่องมือนี้ช่วยลดเวลาในการพัฒนาและทำให้หน้าเว็บมีฟังก์ชันที่ครบถ้วนและใช้งานง่าย.

Assignment 2

VueJS
ออกแบบและสร้างหน้าเว็บแบบทดสอบด้วย Vue.js และ Vuetify พร้อม Responsive Web Design ทำโดยใช้ Vue.js สำหรับการจัดการ DOM, Vuetify สำหรับคอมโพเนนต์ UI ที่รองรับ Material Design, และใช้ CSS media queries หรือ Flexbox/Grid สำหรับการทำให้เว็บแสดงผลได้ โดยเริ่มจากหน้าเข้าสู่ระบบ (Home), หน้าแบบทดสอบ (Quiz), และหน้าผลลัพธ์ (Results) ที่ปรับตัวได้ตามอุปกรณ์.

Assignment 3

ReactJS
ReactJS เป็นไลบรารีที่ได้รับความนิยมอย่างมากในการพัฒนาเว็บแอปพลิเคชัน เนื่องจากมันช่วยให้การสร้างส่วนติดต่อผู้ใช้ (UI) เป็นเรื่องง่ายและมีประสิทธิภาพ ReactJS ช่วยให้การจัดการกับส่วนต่าง ๆ ของแอปพลิเคชันเป็นไปอย่างมีระเบียบ ด้วยการแบ่ง UI เป็นคอมโพเนนต์ที่สามารถนำกลับมาใช้ใหม่ได้ ซึ่งทำให้การพัฒนาแอปพลิเคชันขนาดใหญ่และซับซ้อนเป็นไปได้อย่างราบรื่น นอกจากนี้ ReactJS ยังมีระบบการจัดการสถานะ (state management) ที่มีประสิทธิภาพ

Assignment 4

A-Frame , VR
A-Frame และเทคโนโลยี VR (Virtual Reality) เป็นเครื่องมือที่ช่วยให้ผู้พัฒนาสามารถสร้างประสบการณ์เสมือนจริงในรูปแบบ 3D ได้อย่างง่ายดายและมีประสิทธิภาพ A-Frame เป็นเฟรมเวิร์กที่ทำให้การสร้างแอปพลิเคชัน VR เป็นเรื่องที่สามารถเข้าถึงได้ง่าย ด้วยการใช้ HTML และ JavaScript ซึ่งช่วยให้การพัฒนาโลกเสมือนจริงไม่ซับซ้อนเกินไป โดยไม่จำเป็นต้องมีความรู้ด้านกราฟิก 3D หรือการโปรแกรมที่ซับซ้อน A-Frame รองรับการทำงานกับอุปกรณ์ VR ที่หลากหลาย

Assignment 5

React-Native , Expo
React Native และ Expo เป็นเครื่องมือที่ช่วยให้การพัฒนาแอปพลิเคชันมือถือทั้งในระบบ iOS และ Android เป็นเรื่องที่ง่ายและรวดเร็ว โดยใช้ JavaScript และ React ซึ่งเป็นเทคโนโลยีที่นักพัฒนาคุ้นเคยอยู่แล้ว React Native ช่วยให้สามารถสร้างแอปพลิเคชันที่มีประสิทธิภาพสูงได้ในเวลาไม่นาน ด้วยการใช้คอมโพเนนต์ที่เป็น native ทำให้แอปพลิเคชันสามารถทำงานได้ราบรื่นในทั้งสองระบบปฏิบัติการ ในขณะเดียวกัน Expo ก็เป็นเครื่องมือที่ช่วยให้การพัฒนา React Native

Assignment 6

Firebase
Firebase เป็นแพลตฟอร์มที่ครบวงจรสำหรับการพัฒนาแอปพลิเคชันที่ช่วยให้นักพัฒนาสามารถสร้างแอปได้อย่างรวดเร็วและมีประสิทธิภาพ โดยไม่ต้องกังวลกับการจัดการเซิร์ฟเวอร์หรือโครงสร้างพื้นฐาน Firebase มาพร้อมกับฟีเจอร์ต่างๆ ที่สามารถใช้งานได้ทันที เช่น ฐานข้อมูลแบบเรียลไทม์ (Realtime Database), การตรวจสอบผู้ใช้ (Authentication), การจัดการการแจ้งเตือน (Cloud Messaging), และการวิเคราะห์ข้อมูล (Analytics)
